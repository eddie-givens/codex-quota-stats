Bars First, Numbers On Demand

The hover leads with context and quota bars, then offers a clear "Show all numbers" command link. In VS Code this would be a trusted Markdown command link rather than a native button.

A. Compact Default

Default hover is calm: bars, warning note, and a command link for people who want the raw numbers.

// default hover
numbers are available one click away
warning is informational
